Data vault connectorS: 1.Bottom up, 2.DDL sqlserver
Place the 2 acp files.
In metadata there will be amp file for that we have to create new database and give the login credentials.
Now open the metadata manager, here right click on metadata and create new system for data vault.
After that we have to create new environment.
In data source type -select sql server in drop down.
In DBMS give the created database name,ip address-localhost,login credentials of db and then click on save and exist.
Right click on data env and import the amp file.
small symbol in orange colour at the top right is to-test connection.
When the connection is successful it will generate the mappings in Bottom up connector.


DDL coonector will generate the scripts for metadata.
